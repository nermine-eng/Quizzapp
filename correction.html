La correction du Quizz HTML

  1. HTML c'est quoi ?




    Les réponses correctes sont: HTML est Un langage normalisé de description des pages Web et c'est Hypertext Markup Language


  2. Comment sélectionner une option par défaut dans une liste déroulante en HTML ?




    La réponse correcte est: Utiliser l'attribut "selected" dans la balise "option"


  3. HTML Comment insérer une vidéo YouTube dans une page HTML ?




    La réponse correcte est: Utiliser l'élément "iframe" avec l'URL de la vidéo dans l'attribut src



  4. Comment créer une liste déroulante (select) avec des options provenant d'une base de données ?




    La réponse correcte est:Utiliser PHP pour générer la liste déroulante à partir de la base de données



  5. Comment créer une liste ordonnée numérotée à partir de la valeur 5 avec HTML ?




    La réponse correcte est:ol start="5"
